# Bobyard Takehome Project

Welcome to the Bobyard Takehome project. This repository contains a full-stack application with a React frontend and a FastAPI backend.

## Project Structure

The project is divided into two main directories:

- **`backend/`**: Contains the Python FastAPI server and LanceDB database.
- **`frontend/`**: Contains the React application built with Material UI.

## Getting Started

To run the full application, you will need to set up and run both the backend and frontend services simultaneously.

### Backend
For detailed instructions on setting up and running the backend API, please refer to the [Backend README](backend/README.md).

### Frontend
For detailed instructions on setting up and running the frontend application, please refer to the [Frontend README](frontend/README.md).

## Quick Start Summary

1. **Start the Backend:**
   ```bash
   cd backend
   pip install -r requirements.txt
   uvicorn main:app --reload
   ```

2. **Start the Frontend:**
   ```bash
   cd frontend
   npm install
   npm start
   ```

Access the application at [http://localhost:3000](http://localhost:3000).
